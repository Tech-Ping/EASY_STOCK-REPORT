| (1) 문제 정의 | **[Pain Points]**<br> 1. 어려운 주식 개념 및 용어들: 주식 용어 및 차트 분석이 어려워 스트레스를 받고 실제 투자를 할 수 있을 정도의 지식을 가질 때까지 학습을 이어가기 어려워한다.<br> 2. 손실에 대한 공포: 잘 모르고 아무 데나 투자했다가 돈을 잃게 되는 것에 대한 두려움으로 인해 실제 투자를 시작하기 어려워한다.<br> 3. 시간과 정보 부족: 따로 시간을 내서 공부하지 않으면 신뢰할 수 있는 정보와 쉽게 접근 가능한 자료를 찾기 힘들어 학습 및 투자 의지가 약해진다.<br><br> **[Target Customer]**<br> 주식투자 입문에 진입장벽을 느끼는 2030세대의 남녀<br><br> **페르소나:** 김이화 (23세, 4학년 취준생)<br> - 기존 증권사의 모의 투자 앱을 사용해보았으나 주식 용어가 낯설고, 돈을 잃을까 봐 두려워 삭제함. <br> - 인터넷에 정보가 너무 많아 어디서 시작해야 할지, 무엇이 사실인지 혼란스러워함. <br> - 별도로 주식을 공부하거나 동아리에 참여할 시간을 마련하기 어려움.<br><br> **구체적인 요구:**<br>   - 주식 투자로 돈을 벌고 싶은 욕구: 재정적 자립과 추가 투자 수익을 목표로 자산 관리를 원한다.<br>   - 학습의 용이성: 주식 투자에 대한 두려움을 극복하고 재정적 손실 없는 학습을 원한다.<br>   - 빠르고 간편하게 주식 투자를 배우고 적용할 수 있는 방법을 원한다. |

| (3) 제안 내용 |**SOLUTION 1. AI 챗봇**<br>AI 챗봇 '스토기'와 언제든 자유롭게 대화하며 주식 기초 용어와 지식을 얻을 수 있습니다. 사용자는 주식에 대한 궁금증을 빠르게 해결하고, 주식 투자에 필요한 기초적인 이해를 돕는 다양한 정보를 제공받을 수 있습니다. 이는 기존 서비스에서 제공하지 않는 개인 맞춤형 학습 지원을 통해 초보자들의 이해도를 높이는 데 기여합니다. <br><br>**SOLUTION 2. 모의투자 서비스**<br>사용자는 가상화폐 'STOCKEN'을 발급받아 실제 주식 데이터를 기반으로 한 모의투자 시뮬레이션을 통해 투자 경험을 쌓을 수 있습니다. 이를 통해 실제 시장 환경을 가상으로 체험하며 투자 결정을 연습하고, 실시간으로 주식 데이터를 활용하여 더 나은 투자 전략을 수립할 수 있습니다.<br><br>**SOLUTION 3. 학습 보조 기능**<br>퀘스트를 통해 단계적으로 주식 투자에 대한 이해를 단계적으로 높일 수 있는 학습 기능을 제공합니다. 사용자가 튜토리얼을 마칠 때마다 보상으로 모의 투자용 가상화폐 'STOCKEN'을 발급받을 수 있어, 실제 투자를 위한 준비를 하면서 학습 동기를 부여받을 수 있습니다. 이는 기존 서비스에서 부족했던 학습 동기 부여 요소를 강화하는 기능입니다.|
| (4) 기대효과 및 의의 | **주식 투자 진입 장벽 해소**: AI 챗봇을 이용한 직관적인 학습과 모의 투자 경험으로, 주식 투자에 대한 진입장벽이 줄어들고 더 많은 사람들이 자신감을 가지고 투자에 도전할 수 있게 된다. <br> **개인 맞춤형 투자 교육 제공**: 축적된 데이터를 기반으로 사용자의 투자 성향과 관심사에 맞춘 맞춤형 투자 교육 콘텐츠를 제공하여 학습 효율성을 높이고 실제 투자의 성공 가능성을 높일 수 있다. <br> **모의 투자 데이터를 기반으로 투자 입문자에 대한 인사이트 제공**: 투자 입문자들의 투자 패턴과 성향 관련 인사이트를 통해 향후 금융 교육과 투자 전략 개발에 기여할 수 있다. |
| (5) 주요 기능 리스트    |**1. AI 챗봇 🐱** <br> - 사용자가 주식 용어나 개념에 대해 질문했을 때 빠르고 정확한 답변과 실시간 주식 정보를 제공하여 사용자가 빠르게 의사 결정을 내릴 수 있도록 함<br>**1-1. 일반 질문 답변 (Generative AI)**: GPT-4 Turbo를 Prompt Tuning하여 주식 관련 금융 용어 및 개념에 대한 설명 제공하는 기능<br>**1-2. 주식 데이터 질문 답변 (Yahoo Finance API)**: 사용자가 특정 날짜나 회사의 주식 데이터를 요청하면 Yahoo Finance API를 통해 주식의 시가, 종가, 최고가, 최저가, 거래량 등의 데이터를 제공하는 기능<br> <br>**2. 모의투자 서비스 📊**<br>- 사용자가 다양한 주식 데이터를 실시간으로 조회하고, 이를 바탕으로 투자 결정을 내리며 실제 주식 시장과 유사한 환경에서 모의투자를 진행할 수 있도록 함<br>**2-1. 실시간 주식 데이터 조회 기능**<br>**2-1-1. 실시간 주식 데이터 연동 (한국투자증권 OPEN API)**: Token-based 인증 방식을 사용하여 한국투자증권 OPEN API와 연동하여 실시간 주식 데이터를 가져옴 <br>**2-1-2. 주식 목록 조회**: 전체 주식 목록을 조회하여 각 주식의 현재가, 변동 사항 등을 확인할 수 있도록 구현<br>**2-1-3. 주식 상세 정보 조회 (Alpha Vantage API)**: 특정 주식에 대한 상세 정보를 조회하며, 주가 변동을 캔들차트 형태로 시각적으로 구현<br>**2-1-4. 호가 및 잔량 조회**: 특정 주식의 매수/매도 호가와 잔량을 실시간으로 조회할 수 있도록 구현<br>**2-2. 주식 거래 기능**<br>**2-2-1. 매수/매도 거래**: 사용자가 주식을 매수하거나 매도할 수 있는 거래 기능<br>**2-2-2. 거래 목록 조회**: 진행 중이거나 완료된 거래 목록을 조회할 수 있음<br>**2-2-3. 거래 취소**: 미체결 상태인 거래를 취소할 수 있는 기능<br><br>**3. 학습 보조기능 📚**<br>- 사용자가 단계적으로 주식 투자에 대한 이해도를 높일 수 있도록 튜토리얼과 퀴즈 제공하는 서비스<br>**3-1. 튜토리얼**: 서비스 사용 방법 및 기본적인 투자 방법을 설명하는 레벨별 튜토리얼 구현<br>**3-2. 퀴즈 시스템**: 주식 기초에 대한 문제를 크롤링해와서 레벨별로 분류하고, 랜덤으로 문제를 출제하는 기능 구현<br><br>**4. 투자 리포트 📈**<br>- 이지스톡에서의 모의투자 경험을 바탕으로 사용자의 투자 성향 분석 및 맞춤형 리포트를 제공<br>- Pinecone(Vector DB)를 활용한 벡터 DB 정보와 사용자가 서비스를 사용한 한 달 간의 모의투자 레코드를 트래킹해서 투자 성향 분석 및 그에 따른 정보를 차트로 제공하는 기능<br><br>**5. 주식 뉴스 요약 푸시 알림 📰**<br> - 사용자가 관심 있는 주식에 대한 최신 뉴스를 빠르게 제공하여 투자 결정을 지원하는 서비스<br>**5-1. 최신 주식 뉴스 크롤링 및 요약**: 사용자가 북마크(관심) 표시를 한 Tickers들에 대한 Selenium과 BeautifulSoup을 이용한 네이버 금융 뉴스를 크롤링하여 Langchain 워크플로우로 금융 뉴스 요약 기능 구현<br>**5-2. 주식 뉴스 푸시 알림**: 사용자가 관심 있는 주식 뉴스의 요약본을 FCM을 이용하여 일정 시간마다 푸시 알림을 구현

**RDB (투자 내역 저장)**: 매수/매도 기록, 잔고, 거래 내역 저장<br> - **Client (React)**: 거래 UI, 종목 목록, 상세 차트 등 시각화<br><br><strong>3. 학습 보조 기능 - 초보자를 위한 튜토리얼과 퀴즈 제공</strong><br>튜토리얼은 정적 콘텐츠 기반이며 사용자의 학습 단계 기록을 RDB에 저장. 퀴즈는 크롤러가 수집한 문제를 Spring Boot 서버가 난이도별로 분류하여 API로 전달. 정답 제출 시 정답률을 계산해 학습 진도 관리. 튜토리얼 완료 → 퀴즈 풀기 → 투자 시뮬레이션 연동으로 투자 학습의 흐름을 구성<br>- **Spring Boot 서버**: 퀘스트/퀴즈 로직 처리<br>- **크롤러 모듈 (Selenium + BeautifulSoup)**: 금융 기초 지식 퀴즈 문제 수집<br>- **RDB (퀘스트/퀴즈 저장)**: 퀘스트 진행도, 정답률, 사용자 레벨 기록<br>- **Client (React)**: 단계별 튜토리얼 UI, 퀴즈 풀기 인터페이스<br><br><strong>4. 투자 리포트 -모의투자 내역 기반 투자 성향 분석 및 월간 리포트 제공</strong><br>배치 시스템은 매월 1회 실행되어 사용자의 거래 데이터를 기반으로 성향(리스크 선호도, 분산 투자 여부, 매매 빈도 등)을 분석. 분석된 결과는 Pinecone에 벡터로 저장되어 챗봇이나 다른 기능에서 사용 가능. 시각화된 결과는 React 클라이언트에서 차트로 확인 가능. 리포트는 단순 수익률이 아닌 <em>성향 기반</em>으로 제공되는 점이 특징<br>- <strong>리포트 생성 배치 시스템</strong>: 정기 실행되는 분석/리포트 생성 주체<br>- <strong>Spring Boot 서버</strong>: 리포트 요청 처리 및 전송<br>- <strong>RDB</strong>: 사용자 투자 기록 저장소 (종목, 수익률, 매수/매도 기록 등)<br>- <strong>Pinecone (Vector DB)</strong>: 분석 결과를 벡터화하여 검색 가능하도록 저장<br>- <strong>Client (React)</strong>: <code>react-apexcharts</code>를 사용하여 사용자에게 리포트 시각화 UI 제공<br><br><strong>5. 주식 뉴스 요약 푸시 알림 - 관심 종목에 대한 실시간 뉴스 요약 및 알림 기능</strong><br>Selenium/BeautifulSoup 기반 크롤러가 관심 종목의 최신 뉴스를 주기적으로 수집 → LangChain을 활용해 자연어 요약 처리 → 요약 결과를 벡터화하여 Pinecone에 저장. 설정된 알림 주기에 따라 FCM으로 사용자에게 뉴스 요약을 푸시 알림으로 전송. 사용자는 클라이언트에서 북마크한 종목 기준으로 알림을 설정할 수 있음<br>- <strong>뉴스 요약 모듈 (LangChain + 크롤러)</strong>: 네이버 금융 뉴스 크롤링 및 요약<br>- <strong>Pinecone (Vector DB)</strong>: 요약된 뉴스 벡터 저장 및 검색<br>- <strong>FCM (Firebase Cloud Messaging)</strong>: 푸시 알림 전송<br>- <strong>Spring Boot 서버</strong>: 사용자 관심 종목 관리, 뉴스 요청 분배<br>- <strong>Client (React)</strong>: 뉴스 리스트 및 요약 확인 UI<br> |
